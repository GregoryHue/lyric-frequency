

# Album Word Disparity


## Project setup

Install python libraries with:

```
cd app/
virtualenv --python="/usr/bin/python3" env 
source env/bin/activate
pip install -r requirements.txt
```

## Versions

- Python 3.10.12
- Scrapy 2.11.2
